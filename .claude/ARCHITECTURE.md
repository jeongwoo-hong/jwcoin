```
┌──────────────────────────────────────────────────────────────┐

│                                                              │

│  │ │ (Upbit) │ │ (Upbit) │ │ (pyupbit)│ │ (API)   │   ││
│  │ └────┬─────┘ └────┬─────┘ └────┬─────┘ └────┬─────┘   ││
│  │      │            │            │            │          ││

## AI 프롬프트 구조

```
┌─────────────────────────────────────────────────────────────┐
│                    System Prompt                             │
├─────────────────────────────────────────────────────────────┤

                              +
┌─────────────────────────────────────────────────────────────┐
│                     User Prompt                              │
├─────────────────────────────────────────────────────────────┤
│ • 현재 투자 상태 (JSON)                                      │
│ • 호가 데이터 (JSON)                                         │

│ • 시간봉 OHLCV + 지표 (JSON)                                 │

```

## 환경별 설정

### Local 환경
```
ENVIRONMENT=local
ChromeDriver: webdriver-manager 자동 설치
```

### EC2 환경
```
ENVIRONMENT=ec2
ChromeDriver: /usr/bin/chromedriver (수동 설치)
Chrome 옵션: --headless, --no-sandbox, --disable-gpu
```

```
┌─────────────────────────────────────────────────────────────┐
│                    Error Handling                            │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  API 오류                                                    │
│  ├─ Upbit API 실패 → 로깅 후 종료                            │
│  ├─ OpenAI API 실패 → 로깅 후 종료                           │
│  └─ SerpAPI 실패 → 빈 배열 반환, 계속 진행                    │

│  Selenium 오류                                               │
│  ├─ Timeout → 로깅, chart_image = None                      │
│  ├─ Element 미발견 → 로깅, 계속 진행                         │
│  └─ WebDriver 오류 → 로깅, chart_image = None               │
│                                                             │
│  매매 오류                                                   │
│  ├─ 잔고 부족 → 경고 로깅, 스킵                              │

│                                                             │
│  중복 실행 방지                                              │
│  └─ trading_in_progress 플래그로 제어                        │

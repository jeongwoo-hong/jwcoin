## 1. 코인 자동매매 시스템 (jwcoin)

### 1.1 현재 상태: 6/10점 (수익 어려움)

#### 핵심 문제점

| 문제 | 현재 값 | 영향 | 심각도 |
|------|---------|------|--------|
| **수수료 미반영** | 0.05% (편도) | 실제 0.1% → 수익 잠식 | 🔴 심각 |
| **과도한 거래 빈도** | 일 30회 한도 | 수수료만 3% 손실 | 🔴 심각 |
| **큰 손절선** | -7% | 복구에 7.5% 필요 | 🟠 높음 |
| **트리거 보호 기간** | 15분 비활성화 | 급락 시 손절 기회 상실 | 🟠 높음 |
| **모니터링 간격** | 10초 | 암호화폐는 1-2초 권장 | 🟡 중간 |

#### 수익 시뮬레이션

```
[현재 설정 - 일 20회 거래]
거래 수익: 20회 × 0.3% = +6%
수수료 손실: 20회 × 0.1% = -2%
실제 순수익: +4% (이론)
→ 실제론 손실 가능성 높음 (슬리피지, 불리한 체결 등)

[개선 후 - 일 5회 거래]
거래 수익: 5회 × 0.5% = +2.5%
수수료 손실: 5회 × 0.1% = -0.5%
실제 순수익: +2.0%
→ 월 40-60% 달성 가능
```

#### 즉시 수정 필요 항목

```python
# config/settings.py

# 1. 수수료 수정 (0.05% → 0.1% 양방향)
UPBIT_FEE_RATE = 0.999  # 기존: 0.9995

# 2. 거래 빈도 축소
MAX_DAILY_TRADES = 15   # 기존: 30

# 3. 손절선 축소
STOP_LOSS_L3 = -0.04    # 기존: -0.07
STOP_LOSS_L2 = -0.03    # 기존: -0.05
STOP_LOSS_L1 = -0.02    # 기존: -0.03
```
